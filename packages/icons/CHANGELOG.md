# @rei/c1-icons Changelog

## 0.1.3-alpha.0 - 2026-02-04

- Added React icon components with entrypoint exports.

## 0.1.2-alpha.0 - 2026-02-02

- Prefixed Vue icon component exports with `C1Icon` for consistency.
- Renamed SVG/JSON export paths to `svg/*`, `data.json`, `names.json`, and `meta.json`.

## 0.1.1-alpha.0 - 2026-02-02

- Migrated build/validation tooling to TypeScript with package-local lint/typecheck support.

## 0.1.0-alpha.0 - 2026-02-02

- Added framework-agnostic icon assets with sprite, JSON map, metadata, and JS exports.
- Added SVGO-based optimization for build outputs.
- Added icon validation rules and basic tag/category metadata generation.
